Install in seconds, build in milliseconds.

Hugo works on macOS, Windows, Linux, FreeBSD, and others.

Host on any server or your favorite CDN.

Mac OS

$ brew install hugo

Windows

$ choco install hugo -confirm

Linux

$ snap install hugo
